### Requirements

- **Neovim 0.11.2+**
- **Git**
- **ripgrep** (for search functionality)

### Installation

```bash
mv ~/.config/nvim ~/.config/nvim.backup
git clone https://github.com/NokiaTh131/my-neovim-config.git ~/.config/nvim
nvim
```
